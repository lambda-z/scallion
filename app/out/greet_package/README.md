# Greet Package

A minimal Python package that provides a `greet` function.

## Installation

Install from local directory:

```bash
pip install .
```

Or install directly from a Git repository (after pushing to GitHub/GitLab):

```bash
pip install git+https://github.com/yourusername/greet_package.git
```

## Usage

```python
from greet_package import greet

print(greet('World'))  # Output: Hello, World!
```
